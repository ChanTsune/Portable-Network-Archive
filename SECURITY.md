# Security Policy

## Supported Versions

Portable Network Archive (PNA) is in the `0.x` development series. While in this series, security fixes are applied only to the latest release. Please upgrade to the latest version to receive security updates.

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues or pull requests.

Instead, please report any security vulnerabilities through [GitHub Security Advisories](https://github.com/ChanTsune/Portable-Network-Archive/security/advisories/new) for this repository.

Once we receive your report, we will acknowledge it as soon as possible and keep you informed of our progress toward a fix. Please note that response times may vary, as this project is maintained on a best-effort basis.
