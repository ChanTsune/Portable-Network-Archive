## 2025-05-14 - [Single-Pass Chunk Writing & Async Batching] **Learning:** Reducing data passes from two to one via on-the-fly CRC calculation (`CrcWriter`) improves CPU efficiency. Additionally, batching small sequential async writes into a single `Vec` before calling `.await` reduces runtime overhead and yield points for framing chunks like headers and trailers. **Action:** Use `write_chunk_single_pass_in` for ephemeral chunks and batch small async writes to minimize runtime context switching.
